# Customer Retention & Churn Analysis

## Project Overview

This project focuses on analyzing customer churn and retention patterns using the Telco Customer Churn Dataset. The objective is to identify factors that contribute to customer churn, understand customer retention behavior, and provide actionable recommendations to improve customer loyalty and business performance.

## Objectives

* Analyze customer churn patterns
* Identify key retention drivers
* Measure churn and retention rates
* Study customer lifetime trends
* Generate business insights and recommendations

## Tools Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Power BI

## Dataset

*Telco Customer Churn Dataset*

The dataset contains customer demographics, subscription details, services used, billing information, and churn status.

## Analysis Performed

### Data Cleaning

* Handled missing values
* Converted TotalCharges to numeric format
* Prepared data for analysis

### Exploratory Data Analysis

* Customer Churn Distribution
* Contract Type vs Churn

* Customer Lifetime (Tenure) Analysis

### Dashboard Development

An interactive Power BI dashboard was created to visualize:

* Total Customers
* Churned Customers
* Churn Rate
* Retention Rate
* Contract Type Analysis
* Payment Method Analysis
* Internet Service Distribution
* Customer Lifetime Trends

## Key Findings

* Churn Rate: 26.54%
* Retention Rate: 73.46%
* Month-to-Month contracts have the highest churn rate.
* Electronic Check users are more likely to churn.
* Fiber Optic customers account for the largest share of churned customers.
* Customers with shorter tenure are more likely to leave.
* Long-term contracts significantly improve customer retention.

## Business Recommendations

1. Promote annual and two-year subscription plans.
2. Improve onboarding for new customers.
3. Introduce customer loyalty and reward programs.
4. Offer personalized retention campaigns.
5. Encourage automatic payment methods.
